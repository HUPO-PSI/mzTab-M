= mzTab-M Documentation Hub
:description: The official documentation for mzTab-M, the PSI standard for reporting MS-based metabolomics and lipidomics results.

mzTab-M is the https://psidev.info[HUPO-PSI] community standard for reporting mass spectrometry-based small molecule metabolomics and lipidomics results.
It is designed to be compact, human-readable, and machine-parseable, enabling straightforward integration with downstream analysis pipelines and data repositories.

== Highlights

[cols="1,3",frame=none,grid=none]
|===
|*Semantically enriched*
|All identifiers, units, and controlled vocabulary terms are referenced via ontology entries from https://www.ebi.ac.uk/ols/ontologies/ms[PSI-MS], https://www.ebi.ac.uk/ols/ontologies/chebi[ChEBI], and other community ontologies.

|*Typed and structured*
|Mandatory and optional fields are formally typed. The xref:developers:schema.adoc[JSON schema] and xref:developers:specification.adoc[specification] define every allowed value unambiguously.

|*Validation profiles*
|Results can be validated against xref:validation:profiles.adoc[tiered validation profiles] that reflect the level of identification confidence, from feature-only through summary-level reporting.

|*AI-ready*
|Structured, ontology-anchored metadata makes mzTab-M files suitable for automated processing and integration into AI/ML pipelines without additional curation.

|*MetaboLights submission*
|mzTab-M can be used to create full local submissions in ISA-Tab and MAF format for https://www.ebi.ac.uk/metabolights/[MetaboLights], the EMBL-EBI repository for metabolomics experiments, using the https://github.com/EBI-Metabolights/mztabm2mtbls[mztabm2mtbls] library.
|===
